# Gestor de Gastos Personales

Aplicación web desarrollada con React y Vite para registrar ingresos y gastos, y visualizar el balance en tiempo real.

## Demo

https://martinovalentin96-cmd.github.io/TP2---GestorGastos/

## Funcionalidades

- Registro de ingresos y gastos
- Cálculo automático del balance
- Edición y eliminación de movimientos
- Asignación y filtro por categoría
- Ordenamiento por monto y por orden de carga
- Registro de fecha de cada movimiento
- Límite de gastos con alerta visual

## Tecnologías

- React
- Vite
- JavaScript
- CSS
